Finds the last item in the list that satisfies the condition defined by predicate. If no item matches the condition, than the default value for T (null or all-zero) is returned.

The last item that satisfies the condition predicate. If no item satisfies that condition, the default value for T is returned.
Remarks
If the default value for T (null or all-zero) matches the condition defined by predicate, and the list might contain the default value, then it is impossible to distinguish the different between finding the default value and not finding any item. To distinguish these cases, use TryFindLast(PredicateT, T).
